# Créditos y Atribuciones

El desarrollo de este proyecto es posible gracias a las siguientes herramientas de código abierto y recursos oficiales:

* **Supervision:** Desarrollado por Roboflow. Utilizado para el manejo unificado de detecciones, zonas de conteo y anotaciones visuales.
* **Ultralytics (YOLO & SAM 3):** Herramientas principales para la inferencia y generación de máscaras de segmentación.
* **OpenCV:** Fundamental para el procesamiento frame a frame de los videos y el manejo matricial de imágenes.
* **ByteTrack (trackers):** Algoritmo utilizado para la asociación de detecciones y asignación de identificadores (IDs) a través del tiempo.
* **Material Oficial Copa FutBotMX:** Los videos de prueba utilizados para alimentar los modelos pertenecen al repositorio oficial de la competencia, proveídos por la Federación Mexicana de Robótica.
